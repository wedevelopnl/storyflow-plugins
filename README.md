# StoryFlow Marketplace

A [Claude Code](https://docs.anthropic.com/en/docs/claude-code) plugin marketplace by [WeDevelop](https://wedevelop.nl).

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [storyflow](./storyflow/) | Browse briefings, claim work, and generate implementation plans from client requirements |

## Quick Start

### 1. Add the marketplace

```
/plugin marketplace add wedevelopnl/storyflow-marketplace
```

### 2. Install a plugin

```
/plugin install storyflow@storyflow-marketplace
```

### 3. Authenticate

```
claude mcp auth storyflow
```

See each plugin's README for detailed setup and usage instructions.

## About StoryFlow

StoryFlow enables Software Development as a Service: a platform where development agencies offer transparent, value-based software development to their clients.
